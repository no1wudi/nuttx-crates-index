NuttX Crates Index

Understanding the Table

This table shows the memory usage impact of each Rust crate on the NuttX firmware size. Values represent the difference between builds with and without the crate enabled:

  • Text Δ: Increase in code/executable memory usage
  • Data Δ: Increase in initialized data memory usage
  • BSS Δ: Increase in uninitialized data memory usage
  • Total Δ: Total memory footprint increase
  • Test Result: Status of the crate's test suite, click to view output

Values are shown in bytes, with percentage representing the proportion of memory added relative to the baseline build (without the crate).
